Project Overview

With climate change occuring accross British Columbia, tree mortality rates across the province have increased, specifially mortalities caused by drought. This project develops an automated workflow using QGIS Model Builder to assess tree drought mortalities across BC. The model combines multiple spatial datasets to identify areas with mortalites, enabling awareness of most affected regions by climate change.


By automating the analysis through Model Builder, this workflow can be easily updated with new data, making it a scalable solution for ongoing tree mortality analysis.

QGIS Model Builder Workflow

The Tree Drought Mortalities automated model processes multiple spatial datasets through a series of geoprocessing steps, applying summarize overlay analysis to generate a comprehensive risk assessment. This approach ensures consistency, repeatability, and enables rapid updates when new data becomes available.
